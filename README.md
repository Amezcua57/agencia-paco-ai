# 🤖 Agencia Paco AI

Plataforma completa de marketing con IA para gestionar múltiples negocios desde un solo dashboard.

## ¿Qué es esto?

Agencia Paco AI es un sistema de agencia de marketing 100% potenciado por inteligencia artificial que permite a Francisco Amezcua administrar múltiples negocios, campañas, leads y contenido desde un único panel de control.

## Negocios registrados

| Negocio | Tipo |
|---|---|
| 🐚 Caracol Diamante | Inversión / Bienes Raíces |
| 🛡️ Seguros y Fianzas | Seguros |
| 🏠 Bienes Raíces | Inmobiliaria |
| 📣 Campañas Personales | Marketing personal |

## Módulos incluidos (MVP)

- **Dashboard** — métricas, cola de autorizaciones, actividad reciente
- **Cerebro Vivo** — base de conocimiento por negocio
- **CRM Kanban** — pipeline de leads en 6 etapas
- **Campañas** — con Pre-Autorizador IA
- **24 Agentes de IA** — con ficha técnica y nivel de autonomía 1-5
- **Autorizaciones** — cola de aprobación para acciones sensibles
- **Configuración** — negocios, usuarios, integraciones, modelos IA, reglas
- **Mensajero flotante** — para preguntas e ideas en toda la plataforma

## Stack técnico

### MVP actual
- HTML + CSS + JavaScript (single file, sin dependencias)

### Producción (próxima fase)
- **Frontend:** Next.js 14 + React + TypeScript + Tailwind + shadcn/ui
- **Backend:** Node.js + Fastify + Google Cloud Run
- **Base de datos:** PostgreSQL + pgvector (Supabase)
- **IA:** Claude API (Anthropic)
- **Deploy:** Vercel + Supabase
